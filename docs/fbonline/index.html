# FlippingBook Online API

FlippingBook Online is a service for converting PDFs into interactive digital documents. Its API allows you to programmatically create publications and customize their looks and behavior.

Publications are represented as Publication entities which are build of one or more Source entity. Source represents one publication version and usually corresponds to one source PDF file. To make things even more complex each publication may contains zero or many Tracked/Individual Links entities which represent special links to publication with separate opening tracking, statistics and notifications.

To ease external systems integration there's Publication Triggers and Account Wide Webhooks.

To allow easier API exploration most entities contain HATEOAS links.
